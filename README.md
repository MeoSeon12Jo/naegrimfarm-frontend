## 1. 프로젝트 주제
### 유화 제작 프로젝트
```
AI가 사용자가 업로드한 사진을 유화처럼 만들어주는 프로그램입니다.
이후, 생성된 그림을 경매에 올려 사용자들끼리 포인트로 사고 팔 수 있습니다.
```
